Visual studio Code: Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages and runtimes.

Getting up and running with Visual Studio Code is quick and easy. It is a small download so you can install in a matter of minutes and give VS Code a try.
Cross platform: VS Code is a free code editor, which runs on the macOS, Linux, and Windows operating systems.

Steps to start using VS Code: Open VS Code< File< Open Folder< Open the folder that you have saved in your desktop/laptop< Right click on the empty space and name your file (ex: index.html)< start coding !

(To view what exactly my project is: Right click on your device and click on INSPECT. Or, press "ctrl+shift+I".)
